# Guilds of Verra

## Vision

Guilds of Verra is a progression and adventuring system for Minecraft Fabric.

Our goal is to make every adventure meaningful. Players improve by fighting,
mining, travelling, farming and surviving—not by repeatedly grinding the same
action.

## Design principles

- Every adventure matters.
- Progression should reward active gameplay.
- Avoid unnecessary and repetitive grinding.
- Important levels should unlock meaningful content.
- Exploration should never become impossible because of bad RNG.
- Players should always understand what they can unlock next.
- Skills should encourage different playstyles and cooperation.
- Configuration should be accessible to modpack creators.

## Initial skills

- Fighting
- Mining
- Adventuring
- Farming
- Survival

## First playable milestone

Alpha 0.1 — The First Steps

- Fabric 1.21.1 project builds successfully.
- Puffish Skills integration loads.
- Players can earn Mining experience.
- Basic progress is saved.
- A simple Guild Journal opens with the J key.
