# Guilds of Verra

Guilds of Verra is a progression and adventuring system for Minecraft Fabric 1.21.1.

The project combines skill progression, equipment restrictions, dimension unlocking, guild ranks, discoveries, objectives, and an in-game Guild Journal.

## Planned Skills

- Fighting
- Mining
- Adventuring
- Farming
- Survival

## Core Design Principles

- Progression should reward normal gameplay.
- Exploration should remain useful throughout the game.
- Important unlocks should feel earned.
- Players should always understand their next goal.
- Repetitive grinding should not be the optimal strategy.
- Multiplayer progression should remain fair for every player.

## Planned Features

- Activity-based skill experience
- Puffish Skills integration
- Tool, weapon, and armor requirements
- Nether and End progression
- Guild ranks and promotions
- Structure and biome discoveries
- Adventure objectives
- Guild Journal interface
- Configurable balancing
- Multiplayer support

## Target Platform

- Minecraft 1.21.1
- Fabric
- Java 21
- Puffish Skills 0.18.1

## Status

Early design and development.
